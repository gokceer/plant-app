# Plant App ☘️

A mobile case study app built for **HubX** using **React Native** with Expo.

## ✨ Features

- Onboarding flow
- Home screen with tab navigation

## 🚀 Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

3. Open on device

   Scan the QR code using the Expo Go app.

## 🧪 Tests

```bash
npm run test
```
